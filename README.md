# What is AltAdder?
AltAdder is an Online AltSource (AltStore Sourxe) viewer inspired by AltDirect
# What makes AltAdder unique?
AltAdder shows a preview of the source, which means it can actually only load AltStore Sources that actually exist rather than what AltDirect did.
# How do i use it?
enter a source url or press a trusted source to view it's info
Tap your sideloader on the source page to open your sideloader to add the source
# How should i be viewing the tool?
You should be on an iPhone using Safari to view this.
